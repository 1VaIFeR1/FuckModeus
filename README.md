
Легковесный Android-клиент для API расписания schedule.rdcenter.ru.

Функционал
  Кеширование: Загруженное расписание сохраняется локально для офлайн-доступа.
  Поиск: Быстрый поиск по локальной базе объектов (преподаватели, группы, аудитории).
  Закрепленные: Возможность добавлять объекты в список быстрого доступа.
  
Навигация:
  Горизонтальная прокрутка недель и дней.
  Свайпы влево/вправо для переключения между днями.
  
UI:
  Принудительная тёмная тема.
  Отображение полной сетки занятий с "пустыми" парами.
  Полноэкранный режим.
  Детальная информация о паре по долгому нажатию с возможностью перехода к связанным расписаниям.
  
Стек технологий
  Язык: Kotlin
  Архитектура: MVVM
  UI: Android Views (XML), Material Components, RecyclerView
  Сеть: Retrofit2, OkHttp, Gson
  Асинхронность: Kotlin Coroutines
  Локальное хранилище: SharedPreferences
  
Сборка
  Клонируйте репозиторий.
  Откройте проект в Android Studio.
  Поместите файл allid.txt в директорию app/src/main/assets/.
  Файл allid.txt должен быть валидным JSON-массивом объектов формата {"name": "...", "person_id": "..."}.
  Запустите сборку.

Ну или скачай из релизов.
