FuckModeus

Альтернативный клиент для просмотра расписания ЮФУ (система Modeus). Приложение обеспечивает быстрый доступ к расписанию студентов, преподавателей и аудиторий с возможностью работы без подключения к интернету.

Возможности

 - Два режима работы: Поддержка официального  SFEDU Modeus (с авторизацией через Microsoft) и альтернативного сервера ИКТИБ Расписания (без авторизации).

 - Оффлайн-доступ: Автоматическое сохранение загруженного расписания в память устройства для просмотра без интернета.

 - Удобная навигация: Переключение между днями осуществляется горизонтальным свайпом, как перелистывание страниц, а также касанием.
 
 - Мультипрофили: Возможность создания и переключения между несколькими (до 10) независимыми профилями расписания.

 - Поиск и Избранное: Быстрый поиск по базе преподавателей, учеников и аудиторий с возможностью закрепления часто используемых объектов. Базу можно обновить прямо в приложении (Требует выбранный источник SFEDU Modeus).

 - Адаптивность: Автоматическое определение текущего семестра и учебной недели на основе даты.

 - Гибкая настройка: Возможность ручного ввода адреса сервера для режима "ИКТИБ Расписания" в случае изменения домена.


Использование

Для начала работы необходимо открыть боковое меню, воспользоваться поиском и выбрать интересующий объект (группу, преподавателя или аудиторию). При выборе источника SFEDU потребуется единоразовая (токен действителен сутки!) авторизация через учетную запись университета.

Требования

Android 7.0 и выше.
